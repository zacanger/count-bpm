# count-bpm

Tiny CLI for getting an approximate BPM

--------

## Usage

`npx count-bpm`

Tap on the enter/return key for a bit. Hit `ctrl+d` when you're done.

You can also install globally if that's your thing: `npm i -g count-bpm`
